Sk8M8 - The ultimate skating app for professionals as well as beginners

Projektmedlemmar: Göran, Robert, Joakim, Nikolas

Vi ska göra en skateboard applikation där man kan se hur högt man hoppat i en session och hur snabbt man åker.
Syftet med den applikationen är att hjälpa erfarna skatare så väl som nybörjare att utveckla sina tricks och se vad de behöver
jobba med. I en session som omfattar ca 10 sekunder samlas information från en accelerometer som sitter på skateboarden och som
sen skickas till webbapplikationen. Där får du upp statistik som: 
1. Hur högt du hoppade
2. Hur snabbt du åkte
3. Lägsta/högsta punkt

Utifrån den statistiken ser användaren vad som behövs utvecklas för att nå sina mål att exempelvis sätta en kickflip.
Detta görs alltså med hjälp av Photons och Internetbuttons accelerometer. 

Metod: Kanban

Den hårdvara som krävs:
- Photon
- Internetbutton
- Ett gyro-chip om så behövs

Programmeringspråk: 
- Python med ev. libaries 
- C++ med API från Particle.io
- HTML, CSS


Databas: 
- MySQL
- Python-databas
- Json

Produkt: 
- Webb-applikation
